# Survey Form Project

This project is a simple survey form created using HTML and CSS. The form allows users to provide feedback on various aspects of a fictional service.

## Project Structure

- `index.html`: The main HTML file containing the structure of the survey form.
- `styles.css`: The CSS file for styling the survey form.

## Features

- Input fields for name, email, and age.
- Dropdown menus for selecting roles and preferences.
- Radio buttons for rating recommendations.
- Checkboxes for selecting multiple options.
- Submit button to send the form data.

## Usage

1. Clone the repository or download the project files.
2. Open `index.html` in your web browser to view and interact with the survey form.

## Customization

Feel free to customize the form by editing the HTML and CSS files. You can add more input fields, adjust the styling, or modify the existing elements to suit your needs.
